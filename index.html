üë©‚Äçüíª Developer Quickstart

Use the CollectiVAI Router as a single entry point for multiple AI providers ‚Äì from command line, backends or web frontends. Below are minimal examples for connecting to /api/chat.

Router API ‚Äì cURL example

curl -X POST https://your-router.example.org/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "How can citizens participate in climate decisions?",
    "mode": "ethical",
    "provider": "auto",
    "topic": "climate",
    "modelId": null,
    "serviceProfile": "citizen_advisor"
  }'

SDK ‚Äì pseudo TypeScript client

import { CollectivAIClient } from "@collectivai/sdk";

const client = new CollectivAIClient({
  baseUrl: "https://your-router.example.org/api/chat",
});

const reply = await client.chat({
  prompt: "Draft a policy explainer for citizens...",
  mode: "ethical",
  provider: "auto",
  topic: "democracy",
});

console.log(reply);
